# Hello World 2027

A lightweight mobile countdown, daily unlock log, memory archive, and share-card prototype.

## GitHub Pages 发布

1. Create a new GitHub repository, for example `hello-world-2027`.
2. Upload all files in this folder to the repository root:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.json`
   - `icon.svg`
   - `sw.js`
   - `.nojekyll`
3. In GitHub, open `Settings` → `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select `main` branch and `/ (root)`.
6. Save. The site will be available at:

```text
https://YOUR_USERNAME.github.io/hello-world-2027/
```

## 手机使用

Open the GitHub Pages URL on mobile, then add it to the home screen.

For development preview buttons, open:

```text
https://YOUR_USERNAME.github.io/hello-world-2027/?dev=1
```

Without `?dev=1`, demo buttons are hidden for normal use.
